# Hot Sales

Landing page comercial do Hot Sales.

## Publicação

Este repositório é estático e pode ser hospedado no GitHub Pages com Cloudflare na frente.

Arquivos principais:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Domínio

Quando houver domínio próprio, aponte o DNS no Cloudflare para o GitHub Pages.
